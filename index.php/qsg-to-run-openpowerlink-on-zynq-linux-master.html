openPOWERLINK on Zynq Linux Master

 

The openPOWERLINK master software stack is executed in a Linux environment which runs on the two ARM cores present inside Zynq.

Here the time critical network support layers of openPOWERLINK run as a kernel module inside the Linux environment. The user support library along with a demo application run in the user space.

The design uses the hardware resources available in the processing system (PS) for operation i.e.: 1 Ethernet PHY for POWERLINK communication, built in triple timer counters (TTC) for high resolution timers for openPOWERLINK.
